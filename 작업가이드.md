# 환불금 계산기 — 작업 가이드

## 프로젝트 개요

내일배움카드 수강생 중도포기 시 환불금을 계산하는 단일 HTML 파일 계산기.
설치 없이 브라우저에서 더블클릭으로 실행되며, GitHub Pages로 배포되어 있음.

---

## 파일 구조

```
자부담금계산기/
├── 환불금계산기.html       # 계산기 본체 (데이터 + UI + 로직 모두 포함)
├── 수정방법가이드.md       # 과정 데이터 수정 방법
├── 작업가이드.md           # 이 파일
└── 과정별차시및자부담금.csv # 원본 과정 데이터 (참고용)
```

---

## 기술 스택

- HTML5 + CSS3 + Vanilla JavaScript (ES6+)
- 외부 라이브러리 없음, 인터넷 연결 불필요
- 단일 파일 구조

---

## HTML 파일 내부 구조

```
<head>
  <style>            ← CSS 스타일 전체
</head>
<body>
  레이아웃 HTML      ← 입력 박스(파란) + 결과 박스(초록)

  <script>           ← 과정 데이터 배열 (const courses = [...])
  </script>

  <script>           ← UI 로직 전체
    - DOM 요소 참조
    - resetCourseInfo()     : 과정 정보 초기화
    - populateSelect()      : 드롭다운 목록 채우기
    - courseSelect change   : 과정 선택 시 자부담금/차시 표시
    - completedInput input  : 수강 차시 입력 시 버튼 활성화
    - calcRefund()          : 환불금 계산 함수
    - calcBtn click         : 계산 실행 + 결과 표시
    - resetBtn click        : 전체 초기화
  </script>
</body>
```

---

## 환불금 계산식

```
환불금 = Math.round((자부담금 / 총차시) × (총차시 - 수강차시))
```

원 단위 미만 반올림 처리.

---

## 주요 HTML 요소 ID

| ID | 역할 |
|----|------|
| `courseSelect` | 과정 선택 드롭다운 |
| `feeDisplay` | 자부담금 표시 |
| `sessionsDisplay` | 총 차시 표시 |
| `feeItem` / `sessionsItem` | 자부담금/차시 카드 (empty 클래스로 색상 제어) |
| `completedInput` | 수강한 차시 입력 |
| `calcBtn` | 계산 버튼 |
| `resetBtn` | 초기화 버튼 |
| `resultAmount` | 환불금액 표시 |
| `resultDetail` | 계산 내역 표시 (hidden 클래스로 표시/숨김 제어) |

---

## CSS 클래스 규칙

- `.empty` : 자부담금/차시 카드 또는 결과 금액이 미선택 상태일 때 (회색 텍스트)
- `.hidden` : 계산 내역 영역 숨김 (`display: none`)
- `.formula` : 계산식 표시 줄 (회색, 상단 구분선)

## 과정 데이터 형식

```javascript
const courses = [
  { name: "과정명", fee: 자부담금숫자, sessions: 총차시숫자 },
  ...
];
```

- `option.value`에는 `courses` 배열의 원본 인덱스가 저장됨 (`courses.indexOf(course)`)
- 과정 선택 시 `courses[parseInt(courseSelect.value)]`로 과정 객체를 가져옴
