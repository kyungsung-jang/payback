# 환불금 계산기 — 수정 방법 가이드

## 수정 대상 파일

`자부담금계산기/환불금계산기.html` 파일 하나만 수정하면 됩니다.

---

## 파일 여는 방법

1. `환불금계산기.html` 파일에서 **마우스 오른쪽 클릭**
2. **연결 프로그램 → 메모장** (또는 메모장++) 선택

> ⚠️ 더블클릭하면 브라우저로 열립니다. 반드시 오른쪽 클릭 → 메모장으로 여세요.

---

## 과정 데이터 위치

파일을 열면 위쪽에 아래와 같은 부분이 있습니다:

```
const courses = [
  { name: "출판을 위한 디지털 편집디자인 인디자인 실무", fee: 47440, sessions: 22 },
  { name: "영상 편집 마스터 클래스, 프리미어 프로 베이직부터 레벨업하기", fee: 80390, sessions: 29 },
  ...
];
```

각 항목의 의미:
- `name` : 과정명 (큰따옴표 안에 작성)
- `fee` : 자부담금 (숫자만, 콤마 없이)
- `sessions` : 총 차시 (숫자만)

## 과정 추가하기

목록의 **마지막 항목 아래**에 같은 형식으로 한 줄 추가합니다.

**수정 전:**
```
  { name: "AI로 백전백승 PPT 만드는 스킬", fee: 64380, sessions: 18 },
  { name: "출판까지 완성하는 그림책 기획,제작,작가 활동 올인원 실무", fee: 47440, sessions: 21 },
];
```

**수정 후 (새 과정 추가):**
```
  { name: "AI로 백전백승 PPT 만드는 스킬", fee: 64380, sessions: 18 },
  { name: "출판까지 완성하는 그림책 기획,제작,작가 활동 올인원 실무", fee: 47440, sessions: 21 },
  { name: "새로 추가할 과정명", fee: 50000, sessions: 20 },
];
```

> ⚠️ 주의: 각 줄 끝에 쉼표(`,`)가 있어야 합니다. 마지막으로 추가한 줄에도 쉼표를 붙이세요.

## 자부담금 또는 차시 변경하기

해당 과정의 줄에서 숫자 부분만 수정합니다.

**예시 — 자부담금 변경:**
```
{ name: "ADsP(빅데이터준전문가) 최단기 합격패스", fee: 66530, sessions: 24 },
                                                        ↑ 이 숫자만 변경
```
